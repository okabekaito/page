ポートフォリオ

制作物の紹介

初めまして、文京学院大学4年の岡部海斗です。専攻は経営学なのですが、今年の4月半ばにプログラミングに出会い、勉強するために後期から大学を休学しています。このサイトは今までに作成してきた成果物を紹介するページです。

ポートフォリオとして作成しました。

学習した内容と時間を記録、スケジュールを管理するためのアプリケーションです。モチベーションを維持して勉強し、習慣化させることを目的として作りました。

Udemy で React を学んだため、練習＆自分用アプリとしてバックエンドに Laravel、フロントエンドに React を使って SPA 化して開発しました。

人間関係や仕事のこと、心に抱え込んでいることを吐き出すためのアプリケーションです。いいねで共感を多く得た投稿はランキングのページに載ります。
